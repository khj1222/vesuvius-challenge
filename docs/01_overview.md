# 01 — 대회 개요

**검증: 2026-07-19 · 출처: scrollprize.org**

## 배경

서기 79년 베수비오 화산 폭발로 헤르쿨라네움의 파피루스 두루마리들이 탄화됐다. 물리적으로 펴면 부서지므로 못 연다. Vesuvius Challenge는 이 두루마리를 **고해상도 CT로 스캔한 3D 볼륨**에서, 물리적 훼손 없이 **가상으로 펼쳐(virtual unwrapping)** 표면의 잉크를 검출해 텍스트를 판독하는 것이 목표다.

2023년 Grand Prize로 첫 두루마리 판독에 성공했고, 이후에도 새 스캔·새 두루마리·툴링 고도화가 계속되고 있어 **상시 진행형 챌린지**다.

## 전체 판독 파이프라인 (3단계)

1. **Virtual Unwrapping (가상 펼침)** — 말려있는 파피루스 층을 CT 슬라이스들 전체에서 추적해 하나의 평면으로 편다.
2. **Surface Reconstruction / Autosegmentation** — 파피루스 한 겹을 수백 바퀴에 걸쳐 자동으로 따라가는 표면 추적. (가장 어려운 지오메트리 문제, Grand Prize 핵심)
3. **Ink Detection (잉크 검출)** — 펼친 표면 볼륨에서 탄소 잉크가 있는 픽셀을 ML로 검출해 글자를 드러냄. **← 우리 진입점.**

## 우리 포지션

- 진입 트랙 = **Progress Prizes**(월간 롤링 오픈소스 기여상). 큰 상금(Grand/First Letters)은 지오메트리 프론티어라 팀·장기전 — 우리는 여기 직접 겨냥 X.
- 진입 과제 = **Ink Detection**. ML 실무자용, 공식 표현으로 "다운로드→첫 예측까지 주말 하나". RTX 5090으로 충분.
- 산출물이 시각적(잉크 확률맵/펼친 이미지)이라 사용자 시각형 선호와 정합.

## 왜 이 대회를 골랐나 (2026-07-19 판단)

후보 5개(Vesuvius·ADIA Lab·Banana Hacks·DREAM×CACHE·BEHAVIOR) 중 선택. 결정 이유 = **롤링/월간이라 하드마감이 9월 병목 파이프라인(finance 9/7·pokemon 9/14·trace 9/15·arc 9/19·biohub 9/29·agi-3 9/30)을 안 건드림.** ADIA($100k)가 스킬핏은 최고였으나 9/17 크런치 정면충돌 + 크립토 지급으로 후순위. Vesuvius는 "틈날 때 찔러넣는" 무충돌 진입이 핵심 강점.
